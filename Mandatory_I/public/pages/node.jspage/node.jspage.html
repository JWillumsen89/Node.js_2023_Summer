Node.js - setup
Node.js is and open-source, cross-platform runtime environment that will let you execute Javascript outside of a web browser. Built on Chrome's V8 Javascript engine and is used for developing server-side applications. Easy to scale and is used by many large companies such as Netflix, Uber, and Paypal. In Node.js there is also npm which is a package manager that will let you install and manage packages.

Let's Begin to build a Node.js project!
Create Directory for your project
To setup your project you will first have to find a suitable location for the new project. Once you have found a suitable location you can create a new directory for your project and cd to the directory. 


    #First create the directory for your project
    mkdir "my-project"

    #then cd into the directory
    cd .\my-project\
Initialize your project with 'npm init'
Running 'npm init' in your project will create a package.json file that will contain information about your project and its dependencies.

Run 'npm init' in your project directory and follow the prompts to create your package.json file. 


    #Start the npm init process by typing in the following command
    npm init

    #You will now be prompted some questions about your project and have to fill in the information
    This utility will walk you through creating a package.json file.
    It only covers the most common items, and tries to guess sensible defaults.

    See `npm help init` for definitive documentation on these fields
    and exactly what they do.

    Use `npm install ` afterwards to install a package and
    save it as a dependency in the package.json file.

    Press ^C at any time to quit.
    #Here you set the name of your project
    package name: (test)
    #And here the version number of your project
    version: (1.0.0)
    #Here you can write a description of your project
    description:
    #Here you can set the entry point of your project - Most projects it will be app.js, but this is all about preference
    entry point: (index.js)
    #Here you can add test commands for your project
    test command:
    #Here you can set the git repository for your project
    git repository:
    #Here you can set the keywords for your project
    keywords:
    #Here you can set the author of your project
    author:
    #Here you can set the license for your project
    license: (ISC)

package.json File
You will now have a package.json file in your project directory that will contain information about your project and its dependencies.

Here is an example of a package.json file: 


    { 
        "name": "Project",
        "version": "0.0.1",
        "description": "Project",
        "type": "module",
        "main": "app.js",
        "scripts": {
            "test": "echo \"Error: no test specified\" && exit 1",
        },
        "repository": {
            "type": "git",
            "url": //Your url could look something like this: "git+https://github.com/JWillumsen89/Node.js_2023_Summer.git"
        },
        "author": "Author Name",
        "license": "Desired License",
        "bugs": {
            "url": //Your url could look something like this: "https://github.com/User/Node.js/issues"
        },
        "homepage": //Your url could look something like this: "https://github.com/User/Node.js#readme",
        "dependencies": {
        }
    }
Install dependencies
You will now need to install the needed dependencies for your project. This can be done in multiple ways. Here I will show some examples of how to install the Express.js dependency. 

    //In your package.json file you can add the dependencies like this and then run 'npm install'.
    { 
        "name": "Project",
        "version": "0.0.1",
        "description": "Project",
        "type": "module",
        "main": "app.js",
        "scripts": {
            "test": "echo \"Error: no test specified\" && exit 1",
        },
        "repository": {
            "type": "git",
            "url": ""
        },
        "author": "Author Name",
        "license": "Desired License",
        "bugs": {
            "url": ""
        },
        "homepage": "",

        //This will install the latest version of Express.js when you run the command 'npm install' or 'npm i', the 'i' is short for 'install'.
        "dependencies": {
            "express": "^4.17.1"
        }
    }
                              
                        


    #Here you will simple run the command 'npm install' or 'npm i' to install the dependencies in your package.json file.
    npm install

    #Or
    npm i
                        
                                    


    #A dependency can also be installed by running the following command:
    npm install express

    #Or
    npm i express

    #Or
    npm i express@4.17.1

    #All of these will add the Express.js dependency to your project or other dependencies if you change the name.
Next step will be to create a file called app.js in your project directory.
Create app.js and run files with node.js
I this example I will show how to create the app.js file and run it with node.js.

First create the app.js file in your project directory and add the following code to the file. 


    #For creating the app.js file, write this line in the terminal.
    touch app.js

    #This creates the app.js file in your project directory.
Now you have a app.js file where you can write Javascript code in it and run the file with node.js.

Here is an example of a very simple app.js file: 


    const name = 'John';

    function sayHi() {
        console.log('Hi, ' + name);
    }

    sayHi();
                              
                        


    #To run the app.js file with node.js, write this line in the terminal.
    node app.js
    
    #Output: Hi, John
