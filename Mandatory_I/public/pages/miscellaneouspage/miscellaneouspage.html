Working with Git in the terminal
Git is a version control system that allows you to keep track of your changes and work with other developers on the same project. It is a very useful tool that you will need to learn how to use. In this section we will go through the basics of Git and how to use it in the terminal.
Setup Git in your project
First you have to install the .git file in your project directory. This can be done by running the following command in your project directory: 


    #To initialize git in your project directory run the following command
    git init
Pushing to your repository
Next step will be to add all your files to the staging area. Commit your changes and push them to your repository. 


    #To add all your files to the staging area run the following command
    git add -A 
    #The -A flag will add all your files to the staging area. 
    #You can also add files individually by typing in the file name instead of the -A flag.

    #To commit your changes run the following command
    git commit -m "Your commit message"

    #Link your local repository to your remote repository by running the following command
    git remote add origin your-repo-url.git

    #To push your changes to your repository for the first time run the following command
    git push -u origin main

    #After that you can just run the following command to push your changes to your repository
    git push

    #You can also pull the latest changes from your repository by running the following command
    git pull

    #If you want to reset the staging area to the last commit run the following command
    git reset
Other useful Git commands
These are some other useful Git commands that you might need. 


    #To clone a repository run the following command
    git clone your-repo-url.git

    #You can show all the branches for your repository by running the following command
    git branch

    #You can create a new branch by running the following command
    git branch branch-name

    #You can switch to a branch by running the following command
    git checkout branch-name

    #You can delete a branch by running the following command
    git branch -d branch-name

    #You can merge a branch into your main branch by running the following command
    git merge branch-name

    #Fetching will download the latest changes from your repository without merging them into your local files
    git fetch
