URL Terminology
URL terminology is a set of terms that are used to describe the different parts of a URL. This is useful when you are trying to understand how a URL works.
What an URL consist of
An URL consist of the following parts:
  • Protocol
  • Domain
  • Endpoint
  • Subdomain
  • Top-Level Domain - TLD
  • Query String
  • Path Variable


In this example I will dissect the following URL: https://www.example.com/api/users?name=John&age=20 


    https://www.example.com/api/users?name=John&age=20
    //Protocol is the first part of the URL - https://
    //Domain is the second part of the URL - www.example.com
    //Endpoint is the third part of the URL - /api/users
    //Subdomain is the fourth part of the URL - www
    //Top-Level Domain - TLD is the fifth part of the URL - .com
    //Query String is the sixth part of the URL - ?name=John&age=20
    //In the query string name is the key and John is the value

    https://www.example.com/api/users/1
    //Path Variable is the seventh part of the URL - In this example the user with the id 1 is the path variable
